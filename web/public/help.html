Particle Simulator

A C++20 2D particle simulator with a native desktop renderer, a C++ web server, and a React-based browser UI for building scenarios and streaming live simulation snapshots.

Key Features

  • Native C++ simulation core shared by desktop and web experiences
  • JSON scenarios for particles, forces, bounds, and obstacles
  • Live browser playback over WebSocket
  • Scenario builder for authoring and editing setups in the browser
  • Pause, reset, step, and speed controls

Desktop Controls

  • Space: pause or resume
  • R: reset to the initial state
  • N: single simulation step while paused
  • Esc: close the window

Build

cmake -S . -B build
cmake --build build --config Release

Run Desktop App

.\build\Release\particle_simulator.exe .\scenarios\minimal_zero_gravity.json

Web Frontend Setup

cd .\web
npm install
npm run build

Start Web Server

cd ..
.\build\Release\particle_simulator_web_server.exe --web-root web/dist --scenario-dir scenarios

Web API

  • GET /api/scenarios lists bundled example scenarios
  • POST /api/session creates a simulation session from scenario JSON
  • GET /ws?sessionId=... streams snapshots and accepts playback commands

Supported WebSocket commands include play, pause, reset, step, and setSpeed.

Scenario Schema

A valid scenario defines particle types, spawn groups, and geometry. Optional sections include window, simulation, and forces.

  • window: rendering size, title, background, FPS
  • simulation: timestep, seed, collision iterations, grid size
  • forces: gravity, drag, wind, and radial fields
  • particleTypes: radius, mass, restitution, color, initial velocity
  • spawnGroups: particle counts plus spawn position and velocity ranges
  • geometry: world bounds and rectangle/circle obstacles
